This program aims to automate restaurant menu entries using AI. When an image is posted, it is processed by an OCR that transcribes it into text. The text is sent to a generative AI (Gemini), which processes the text based on the context of the prompt and returns it in JSON string format. The JSON string is processed and sent to the frontend for implementation.

ACCEPTS ONLY PNG, JPG, JPEG
